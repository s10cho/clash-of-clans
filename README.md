# 솔로 클래시 (Solo Clash)

혼자서 즐기는 마을 건설 · 약탈 · 클랜전 게임. **완전 오프라인 싱글플레이 PWA**입니다.

클래시 오브 클랜의 규칙(자원/건설/훈련/약탈/클랜전)을 따르되, 서버 없이 브라우저에서만 동작합니다.
모바일 홈 화면에 설치하면 네트워크 없이도 플레이할 수 있고, 모든 진행 상황은 기기(IndexedDB)에 저장됩니다.

## 플레이

GitHub Pages 배포: `https://s10cho.github.io/clash-of-clans/`

**모바일 설치 (PWA)**
- **iOS Safari**: 공유 버튼 → "홈 화면에 추가"
- **Android Chrome**: 메뉴 → "홈 화면에 추가" (또는 설치 배너)

## 게임 규칙 (원작 규칙 기반)

- **자원**: 골드 / 엘릭서 / 다크 엘릭서(TH7+) / 젬. 수집기가 시간당 생산(오프라인에도 누적), 저장고 용량 제한.
- **건설**: 건설업자(오두막 수 = 동시 작업 수)가 건물을 짓고 업그레이드. 타운홀 레벨이 건물 수/레벨 상한 결정.
- **군대**: 훈련소에서 유닛 훈련(훈련소 수만큼 병렬), 군대 캠프 수용량 제한, 연구소에서 유닛 레벨 업그레이드, 주문 공장.
- **약탈**: 상대 검색(골드 비용) → 절차 생성된 적 기지 공격. 별(50% 파괴/타운홀/100%), 저장고 20%·수집기 50%·다크 5% 약탈, 타운홀 레벨 차 페널티, 트로피 증감.
- **방어**: 자리를 비우면 습격당할 수 있음(부재중 시뮬레이션). 파괴율에 따라 실드 획득, 공격하면 실드 해제.
- **클랜전**: 5 vs 5 (나 + AI 동료 4명). 원작 규칙대로 트로피가 아닌 **전력(war weight)** 기반 매칭, 멤버당 공격 2회, 별 수로 승부, 승리 시 보너스 자원. AI 멤버들의 공격은 전쟁일 동안 자동 진행. 빠른 전쟁(10분+2시간) / 정식 전쟁(23시간+24시간) 선택 가능.
- **유닛 AI**: 원작 스타일의 "최근접 타겟 + 성벽 파괴 가중치 A*" — 자이언트/해골 비행선은 방어 시설, 고블린은 자원(2배 피해), 해골 돌격병은 성벽(40배), 호그 라이더는 성벽 점프.

밸런스 수치는 클래식(2015년경) 클래시 오브 클랜 근사값이며 타운홀 1~10을 지원합니다.

## 개발

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 타입체크 + 프로덕션 빌드 (dist/)
npm run icons    # PWA 아이콘 재생성
```

- **스택**: Vite + TypeScript + Canvas 2D (외부 런타임 의존성 없음)
- **렌더링**: 아이소메트릭 절차적 그래픽 — 이미지/사운드 에셋을 전혀 사용하지 않음
- **저장**: IndexedDB (localStorage 폴백), 업그레이드 타이머는 완료 시각 기반이라 오프라인 경과 자동 처리
- **오프라인**: 서비스 워커 — 앱 셸 프리캐시 + 정적 에셋 cache-first + HTML network-first
- **배포**: `main` 푸시 시 GitHub Actions → GitHub Pages (`vite base: './'`로 서브패스 대응)

> 최초 배포 후 저장소 **Settings → Pages → Source를 "GitHub Actions"** 로 설정해야 합니다.

## 법적 고지

이 프로젝트는 개인 학습/비상업 목적의 팬 프로젝트입니다. Supercell의 그래픽·사운드·에셋을 일절 사용하지 않으며(팬 콘텐츠 정책 준수), 모든 그래픽은 절차적으로 생성됩니다. Clash of Clans는 Supercell Oy의 상표입니다. 이 프로젝트는 Supercell과 무관하며, Supercell의 보증이나 후원을 받지 않았습니다.
